# Requirement Assistant

## Project info

Repository: https://github.com/namrathar-18/requirement-assistant

## Setup

1. Install dependencies:

```sh
npm i
```

2. Run the development server:

```sh
npm run dev
```

## What technologies are used for this project?

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
